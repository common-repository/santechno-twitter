display twitter feed on website from your twitter account using the newest Twitter API.
== Description ==
This Plugin use for showing twitter feeds anywhere on your website - it uses widget and you can place it anywhere you want.

== Installation ==

Upload the santechno_twitter plugin to your blog, Activate it, then enter your [twitter api info ).

'twitter_consumer_key' => '',
'twitter_consumer_secret' => '',
'twitter_access_token' => '',
'twitter_access_token_secret' => '',


You're done!
